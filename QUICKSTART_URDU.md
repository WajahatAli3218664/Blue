## انسٹالیشن

```bash
# ڈیپنڈنسیز انسٹال کریں
npm install

# ڈیولپمنٹ سرور شروع کریں
npm start

# پروڈکشن بلڈ بنائیں
npm run build
```

## استعمال

1. **ویب سائٹ کھولیں**: http://localhost:3000
2. **کورس شروع کریں**: "Start Learning" بٹن پر کلک کریں
3. **چیٹ بوٹ استعمال کریں**: نیچے دائیں طرف 🤖 آئیکن پر کلک کریں
4. **ٹیکسٹ سلیکٹ کریں**: کسی بھی ٹیکسٹ کو سلیکٹ کریں اور "Ask AI" پر کلک کریں
